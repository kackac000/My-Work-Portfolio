# 🚀 SAP 출하 업무 자동화 매크로 (SAP Release Tool)

## 📌 프로젝트 개요
물류 현장에서 매일 반복되는 SAP 출하 대사 작업의 소요 시간을 단축하고 휴먼 에러를 방지하기 위해 개발한 윈도우 GUI 기반 매크로 프로그램입니다.

## 🛠 사용 기술 및 환경
* **Language:** Python
* [cite_start]**GUI Framework:** tkinter (상업적 배포 시 라이선스 위반 소지를 차단하기 위해 표준 라이브러리 채택) [cite: 14]
* [cite_start]**Automation:** pyautogui, win32gui [cite: 18, 19]
* [cite_start]**Build:** PyInstaller (단일 파일 배포 최적화) [cite: 24]

## ✨ 핵심 기능 및 기술적 특징
1. [cite_start]**완벽한 포커스 제어 및 안정성:** `GetForegroundWindow()`를 통해 매 루프마다 SAP 창의 활성화 상태를 검증하고, 이탈 시 자동으로 포커스를 복귀시켜 오작동을 방지합니다. [cite: 20]
2. [cite_start]**비동기 멀티스레딩 UI:** 매크로 로직과 UI 업데이트를 분리하여 작업 중에도 프로그램이 멈추지 않으며, 스레드가 안전하게 종료되도록 제어합니다. [cite: 15, 21]
3. [cite_start]**동적 잔여 시간(ETA) 계산:** 고정된 대기 시간이 아닌, 실제 소요된 평균 시간을 계산하여 실시간으로 정확한 남은 시간을 UI에 표시합니다. [cite: 20]

## 💡 기대 효과
* 수작업 대비 업무 소요 시간 대폭 단축
* 반복 작업으로 인한 피로도 감소 및 데이터 처리 정확도 향상
