# 📊 B2B 유통 데이터 분석 및 AI 수요 예측 대시보드

## 📌 프로젝트 개요

SAP 시스템에서 추출한 방대한 유통/물류 데이터(엑셀)를 가볍고 빠른 로컬 DB(DuckDB)로 자동 변환하고,
Streamlit을 활용해 실시간 KPI 분석 및 수요 예측 결과를 시각화하는 웹 대시보드입니다.
물류 현장 실무자의 시각에서 데이터 기반의 빠르고 정확한 의사결정을 지원하기 위해 기획 및 개발했습니다.

---

## 🛠 사용 기술

| 구분 | 내용 |
|------|------|
| **Language** | Python |
| **Database** | DuckDB (대용량 엑셀 데이터를 빠르고 가볍게 처리) |
| **Frontend/Visualization** | Streamlit, Plotly Express |
| **Data Processing** | Pandas |

---

## ✨ 핵심 기능

1. **대용량 데이터 처리 최적화**
   무겁고 느린 엑셀(.xlsx) 원본 데이터를 DuckDB 포맷으로 파싱하여 조회 및 분석 속도를 획기적으로 개선했습니다.

2. **실시간 KPI 시각화**
   복잡한 유통 데이터를 직관적인 차트와 표로 변환하여, 지점별 성과 및 제품/매출처 순위(TOP 30/10)를 클릭 몇 번으로 파악할 수 있습니다.

3. **하이브리드 수요 예측 (영업 기회 분석)**
   최근 3~6개월 평균 실적과 작년 동월 실적을 결합한 하이브리드 알고리즘을 적용하여, 매출처별 추가 발주 가능량과 예상 매출을 자동으로 산출합니다.

4. **결과 리포트 자동 생성**
   분석 및 조회된 데이터를 즉시 엑셀(Excel) 파일로 다운로드하여 현업 담당자에게 바로 전달할 수 있습니다.

## 🤖 AI 협업 개발 (AI-Assisted Development)

본 프로젝트는 물류 현장 실무자의 도메인 지식과 AI(LLM)의 기술적 조력을 결합하여 개발했습니다.

- **아키텍처 설계:** 수백만 건 데이터 처리에 최적화된 DB 선정 및 스키마 설계 과정에서 AI 코드 리뷰 적극 활용
- **로직 고도화:** 하이브리드 수요 예측 알고리즘의 예외 처리 및 비즈니스 로직 정교화
- **보안 강화:** SQL Injection 방지를 위한 파라미터 바인딩 전면 적용 과정에서 AI 피드백 반영
- **UI/UX 개선:** 조건부 하이라이팅 및 데이터 시각화 인터페이스 고도화
