# 📊 엑셀 VBA 업무 자동화 도구 모음 (VBA-Excel-Automator)

15년 물류 현장 경험을 바탕으로, 반복적이고 오류가 잦은 수작업을 **VBA(Visual Basic for Applications)**로 자동화한 실무 밀착형 도구 모음입니다. 
SAP 시스템에서 추출한 데이터를 가공하고, 업로드용 파일을 생성하며, 현장 로케이션 라벨까지 출력하는 전 과정을 자동화하여 **휴먼 에러를 차단하고 업무 소요 시간을 획기적으로 단축**했습니다.

---

## 🔍 배경 및 문제 정의
의약품 물류 현장에서는 거대한 SAP 시스템과 엑셀 사이의 데이터 가공 및 반복 작업이 매일 발생합니다. 수작업에 의존하던 기존 프로세스의 병목을 다음과 같이 해결했습니다.

| 업무 | 기존 방식 (Pain Point) | 자동화 후 개선 (Solution) |
| :--- | :--- | :--- |
| **반품 재고 매칭** | 수백 개 품목을 눈으로 대조하며 수기 입력 | 바코드 스캔 1회로 자동 매칭 및 분류 |
| **SAP 업로드 파일 생성** | 양식에 맞게 데이터를 수동으로 복사/편집 | 버튼 1회 클릭으로 즉시 생성 |
| **대용량 데이터 분할** | 수동 필터링 후 시트 복사 및 개별 저장 반복 | 기준 열(Column) 선택만으로 자동 분할 저장 |
| **재고조사 양식 생성** | SAP Raw Data를 목적에 맞게 수동 가공 | 3가지 타겟 양식 중 선택 후 즉시 포맷팅 |
| **로케이션 라벨 출력** | 라벨 폼에 품목을 하나씩 직접 입력 후 인쇄 | 데이터 범위 지정 후 버튼 1회로 PDF 일괄 생성 |

## 🛠 포함된 도구 및 핵심 기능

### 1. 📦 매입반품 처리 시스템
반품 대상 의약품을 바코드로 스캔하여 현재 재고와 자동 매칭하고, SAP 업로드용 파일까지 즉시 생성합니다.

* **주요 기능:**
  * GS1 바코드에서 표준바코드(13자리), 제조번호(Batch), 유효기한 자동 분리
  * 반품 리스트와 현재 재고를 실시간 비교 및 매칭
  * 매칭 완료 시 자동으로 "완료" 시트로 데이터 이동
  * SAP 업로드 양식 4종(RTP, BT, POI, ksap) 자동 생성
* **처리 흐름 (Workflow):**
  > **[스캔]** 바코드 스캔 (TextBox 입력)
  > **↓**
  > **[파싱]** 표준바코드 / 제조번호 / 유효기한 식별 및 자동 분리
  > **↓**
  > **[매칭]** 반품 리스트와 실재고 시트 크로스 체크
  > **↓**
  > **[분류]** 수량 카운팅 연산 후 완료 시트로 데이터 자동 이동
  > **↓**
  > **[출력]** SAP 업로드 규격에 맞춘 개별 `.xlsx` 파일 자동 생성

### 2. ✂️ 대용량 파일 자동 분할기
대용량 엑셀 데이터를 특정 열(Column)의 값을 기준으로 자동 분할하여 개별 파일로 빠르게 저장합니다.

* **주요 기능:**
  * 기준 셀과 저장 위치를 UI를 통해 직관적으로 지정
  * 지정된 열의 고유 값을 기준으로 데이터를 자동 분할 후 개별 `.xlsx` 저장
  * 중복 파일명 감지 및 덮어쓰기 방지 오류 처리
  * 분할 시 전체 시트 포함 여부(포맷 유지) 선택 가능
* **활용 예시:** 제약사별, 거래처별, 지점별로 데이터 취합본을 쪼개서 개별 송부해야 할 때 수작업 제로화.

### 3. 📋 지능형 재고장 생성기
SAP에서 추출한 무거운 재고 데이터를 정제하고, 목적에 맞는 3가지 재고조사 양식으로 자동 변환합니다.

* **지원 양식:**
  | 양식명 | 용도 및 특징 |
  | :--- | :--- |
  | **전수재고조사 양식** | 연간 대규모 재고조사 시 활용 (담당자, Key-In, 팀장 서명란 자동 포함) |
  | **EzStock 양식** | SAP EWM EzStock 시스템 업로드용 포맷으로 자동 생성 |
  | **일일재고조사 양식** | 일상적인 로케이션별 실물 재고 확인용 경량 포맷 |
* **데이터 정제 로직:** GI-ZONE 로케이션 자동 제거, 제품코드 기준 수량 합산(SUMIF 연산), 중복 데이터 제거 및 로케이션 정렬, 양식별 헤더 및 서식 자동 적용.

### 4. 🏷️ 로케이션 라벨 일괄 출력기
창고 로케이션 라벨을 엑셀 데이터 기반으로 자동 생성하고, 인쇄에 최적화된 PDF 파일로 출력합니다.

* **주요 기능:**
  * 데이터 시트의 제품코드, 제품명, 로케이션을 라벨 템플릿에 자동 매핑 및 배치
  * 페이지당 라벨 수동 조정 가능 (기본 세팅: 2열 × 8행 = 16개)
  * 바코드 폰트 인식 형식(`*바코드값*`) 자동 래핑 적용
  * 생성 당일 날짜가 포함된 파일명으로 바탕화면에 PDF 자동 저장

```text

├── 매입반품_자동화.txt     # 반품 처리 및 SAP 업로드 파일 생성 VBA Source Code
└── 업무도구_자동화.txt     # 파일분할 / 재고장생성 / 라벨출력 VBA Source Code
```

**⚠️ 보안 유의사항:** 실제 현장에서 운영 중인 `.xlsm` 엑셀 파일에는 회사 내부의 민감한 데이터(품목코드, 단가, 거래처 정보 등)가 포함되어 있어 본 저장소에 직접 업로드하지 않았습니다. 대신 핵심 비즈니스 로직이 담긴 VBA 소스 코드만 텍스트 파일 형식으로 안전하게 공개합니다.
