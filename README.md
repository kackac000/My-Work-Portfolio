# My-Work-Portfolio

# 15년차 물류 전문가의 업무 자동화 포트폴리오

물류 현장에서 발생하는 비효율과 반복 업무를 해결하기 위해 직접 개발한 C# 및 VBA 기반의 자동화 툴 모음입니다.

---

## 1. SAP 연동 바코드 스캐너 (C#)

- **개발 동기:** SAP 시스템에 바코드를 스캔시 묶음바코드와 몇몇 바코드의 인식 불가 문제를 해결하고자 개발 했습니다.
- **주요 기술:** C# WinForms, OpenCVSharp (이미지 템플릿 매칭), WinAPI (외부 프로그램 제어)
- **핵심 기능:** 스캔된 바코드 종류(개별, 묶음, OBD 등)를 자동으로 파싱하고, 특정 바코드를 인식 가능하게 변조하여, 이미지 인식을 통해 SAP 화면의 정확한 위치에 값을 입력하고 엔터까지 자동으로 실행합니다.

---

## 2. 엑셀 업무 자동화 스크립트 (VBA)

- **개발 동기:** 매일 반복되는 단순 엑셀 작업을 자동화하여 핵심 업무에 집중할 수 있는 환경을 만들고자 개발했습니다.
- **주요 기능:**
    - **파일 분할:** 특정 기준에 맞춰 하나의 엑셀 파일을 수십 개의 파일로 자동 분할.
    - **재고장/PO 자동 생성:** 원본 데이터를 기반으로 정해진 양식의 재고 조사 리스트 및 PO(발주서)를 자동으로 생성.

*참고: 현업에서 빠른 적용을 목표로 개발한 실무 코드로, 일부 정제되지 않은 부분이 있을 수 있습니다.*
