### 1. 🖱️ SAP ECC→EWM 출하 Release 자동화 매크로
[📂 01_SAP_Release_Automation](./01_SAP_Release_Automation)

| 항목 | 내용 |
| :--- | :--- |
| **Tech** | Python, tkinter, win32gui, pyautogui |
| **수동 작업** | 출하 중 Release 30~40회 × 회당 15초 * 하루 3번 = 약 30분 소요 |
| **자동화 후** | 버튼 1회 클릭으로 전체 자동 처리 → 담당자 개입 시간 거의 0 |
| **연간 효과** | 약 120시간 절감 (연 250일 기준) + 휴먼 에러 제거 |

* SAP VL02N 화면에서 반복 수행해야 하는 출하 Release 작업을 자동화한 Windows GUI 프로그램입니다.
* 멀티스레딩으로 UI 응답성을 유지하며, SAP 창 포커스 자동 감지 및 복구 기능을 내장했습니다.
* 회사 보안 정책(설치 불가) 환경을 고려하여 무설치 단일 EXE로 배포 가능하며, 상업용 라이선스(PSF/BSD)를 완벽히 준수합니다.
